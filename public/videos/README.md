# Vidéo de fond de l'accueil

Dépose ici ta vidéo, avec exactement ces noms de fichiers :

- `hero.mp4` — la vidéo elle-même (recommandé : format vertical/portrait ou 16:9,
  quelques secondes en boucle, sans son puisqu'elle joue automatiquement en muet,
  poids léger si possible pour un chargement rapide sur mobile — quelques Mo max).
- `hero-poster.jpg` — une image fixe (une image extraite de la vidéo par exemple),
  affichée pendant que la vidéo charge.

Aucun fichier ici pour l'instant : sans eux, l'écran d'accueil affichera juste
le fond sombre normal (pas d'erreur, juste pas de vidéo).
